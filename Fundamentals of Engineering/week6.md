## plant layout 工厂布局
设计流程时，尽量减少运输距离，注重效率

越多的距离代表着更多的风险

plant layout drawing 工艺流程图：化学工厂内各工艺单元、管道、设备及设施的物理布局的图形表示。

实现工厂安全、高效和经济的运行，需优化设备、公用设施及基础设施的物理布局

在Process Layout 流程布局中
>设备是按照功能或工艺类型进行分类的
>常见于小批量、分批生产的工厂中
>例如：反应器被集中放置在一个区域，而热交换器则在另一个区域

在Product Layout 产品布局中
>设备按照工艺流程的顺序进行排列
>在大规模生产及连续生产过程中较为常见
>示例：炼油厂或大型化工厂

在Fixed-Position Layout 定位式布置中
>该产品固定在一个位置，而设备或工人则围绕其移动。
>适用于非常大型的产品（例如造船厂、飞机制造厂）

1. 安全事项：确保危险设备之间保持安全距离
2. 效率：减少物料搬运和运输环节
3. 维护：确保便于对设备进行维修和检查
4. 灵活性：便于未来扩展


工厂布局的关键要素
1. 主要工艺单元
>反应器、蒸馏塔、热交换器、储罐
2. 管道
>主工艺管道、辅助管道（例如蒸汽、水、空气）
3. 公用区域
>锅炉、冷却塔、空气压缩机
4. 安全特性
>灭火系统、紧急出口
5. 交通与通行
>道路、维护通道入口
6. 控制室
>便于监测和控制工厂运营的位置

工厂布局的基本原则
1. 安全第一
>保持危险区域与关键区域（例如易燃物品存放区）之间足够的安全距离
>确保良好的通风条件、消防通道以及紧急通道的畅通无阻
2. 减少物料搬运
>物料在各工序之间的移动距离应当尽可能缩短
3. 维护空间
>为设备周围留出足够的空间，以便于操作和维护
4. 未来扩展
>制定未来改造或扩建的计划，同时又不干扰当前的运作
5. 环境考量
>合理设置废物处理设施及排放控制装置

在设计化工厂设备布置时，必须考虑到后期维护的便利性，尤其是换热器、塔、反应器等大型设备的拆装空间和吊装路线。

inherent safety 本征安全
1. Minimize (intensification) 减少（加剧反应）
>核心是减少工艺流程中的停滞时间
2. Substitute (substitution) 替换
>核心是选用合适材料，尽可能减少来自材料的危险
3. Moderate (attenuation and limitation of effects) 适度（影响限制效果）
>限制危险情况发生的环境
4. Simplify (simplification and error tolerance) 简化
>降低操作门槛，设计便于理解

一个本征安全的工厂依靠化学物理原理预防事故，而非控制系统，冗余设计和特殊操作程序等

固有安全的工程能顾容忍一定的错误，具备成本效益

作为防护层中的一部分，固有安全针对工艺设计方面的内容
